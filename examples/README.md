# Example CompScript Files

This directory contains example CompScript files to test the language server.

## Basic Examples

### hello.cs

A simple example showing basic function calls and literals.

### psych-sheet.cs

Example of creating psych sheets with tables and sorting.

### groups.cs

Example of group assignment functionality.

## Testing the Language Server

1. Open any of these files in VS Code
2. Try the following features:
   - Hover over function names
   - Trigger autocomplete with Ctrl+Space
   - See signature help when typing function parameters
   - Check for syntax errors

## Creating Your Own Examples

Create a new file with `.cs` or `.compscript` extension and start coding!
